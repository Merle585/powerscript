# Dansk:
1. Heimdel --> Privilegeret App Control --> Anmod om Admin Rettigheder
2. Åbn PowerShell ISE som administrator
3. Åbn `01_Before_support_Lenovo_dock`, `02_update_Lenovo_dock`, og `03_After_support_Lenovo_dock` i PowerShell ISE
4. Kør `01_Before_support_Lenovo_dock` og gem det i `C:\"01_computer_name_BS".txt`
   - Derefter kør `02_update_Lenovo_dock` og gem det i `C:\"02_computer_name_update".txt`
   - Til sidst kør `03_After_support_Lenovo_dock` og gem det i `C:\"03_computer_name_AS".txt`


# English:
1. Heimdel --> Privileged App Control --> Request Admin Privileges
2. Open PowerShell ISE as administrator
3. Open `01_Before_support_Lenovo_dock`, `02_update_Lenovo_dock`, and `03_After_support_Lenovo_dock` in PowerShell ISE
4. Run `01_Before_support_Lenovo_dock` and save it in `C:\"01_computer_name_BS".txt`
   - Then run `02_update_Lenovo_dock` and save it in `C:\"02_computer_name_update".txt`
   - Finally, run `03_After_support_Lenovo_dock` and save it in `C:\"03_computer_name_AS".txt`
